## Основные параметры <a name="motoparams"></a>

* Объем 249сс для двигателя 172FMM / 270сс для двигателя 175FMM
* Топливо АИ92 / АИ95
* Вес 123 кг (скорее всего, сухой)
* Высота по седлу 86см
* Дорожный просвет 30см
